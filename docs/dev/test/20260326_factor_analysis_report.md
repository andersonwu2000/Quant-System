# 台股量化因子研究總報告

**日期**: 2026-03-26
**報告範圍**: 實驗 1~15 + 66 因子全掃描 + 256 config grid backtest
**總實驗次數**: 1,280+ 回測 + 15 次 IC/Walk-Forward 實驗

---

## 1. 研究概要

對台股進行系統性因子研究，驗證純 price-volume + 公開籌碼面數據是否能穩定產生超額報酬。

| 維度 | 範圍 |
|------|------|
| 因子數 | 66 price-volume + 9 籌碼面 = **75** |
| Universe | 最大 142 支台股（TW50 + 中小型股） |
| 數據期間 | 最長 7 年（2019-01 ~ 2025-12） |
| 持有期間 | 5 / 10 / 20 天 |
| 成本假設 | 單邊 30~50 bps（手續費 0.1425% + 證交稅 0.3%） |

## 6. 三大核心教訓

### 6.1 Selection Bias 是最大陷阱

ivol 在 32 支精選大型股 ICIR 0.60（看似完美），擴大到 142 支即變成 -0.06。**永遠要在最寬的 universe 驗證因子。** PBO 70% 的過擬合警告在後續實驗中完全應驗。

### 6.2 成本是台股 alpha 的終極瓶頸

| 換手率區間 | 因子數 | Viable 數 | 年化成本範圍 |
|-----------|:------:|:---------:|:-----------:|
| 0% ~ 10% | 18 | 4 | 0 ~ 300 bps |
| 10% ~ 30% | 10 | 0 | 300 ~ 900 bps |
| 30% ~ 50% | 12 | 0 | 900 ~ 1500 bps |
| 50% ~ 85% | 26 | 0 | 1500 ~ 2550 bps |

台股單邊成本 ~0.44%（來回 ~0.88%），**換手率 > 10% 的因子幾乎不可能盈利**。Kakushadze 101 短週期因子（為美股設計）在台股全軍覆沒。

### 6.3 1/N 等權在台股極難被打敗

DeMiguel et al. (2009) "1/N" 結論在台股完全驗證。75 個因子、多種分層方式和組合方法，最佳超額僅 +3~6%/年且不穩定。1/N 等權 + 月度再平衡 + DD control 是最穩健的基線。

## 8. 未來突破方向

### 8.1 數據源升級

| 方向 | 預期效果 | 難度 |
|------|---------|------|
| 即時法人數據（非 T+1） | 提前 price in 前獲取信號 | 高（需券商 API） |
| 分析師預期修正 | 學術文獻強因子 | 高（需 TEJ 付費） |
| 基本面（PE/PB/ROE） | 天然低換手率 | 中（部分免費） |
| 融資融券即時餘額 | 散戶情緒指標 | 中 |

### 8.2 方法論改進

| 方向 | 預期效果 |
|------|---------|
| 降低交易成本（券商手續費打折 0.1425% → 0.06%） | 直接改善淨 alpha |
| 市值分層（大/中/小型分開） | 控制 size effect，釋放因子信號 |
| 行業中性化 | 控制行業效應 |
| Regime-aware（牛熊判斷調倉） | 改善熊市 consistency |
| 更長持有期（月頻/季頻） | 降低換手率 → 降低成本 |

### 8.3 模型升級

| 方向 | 預期效果 |
|------|---------|
| Rolling IC 動態加權 | 適應因子輪動 |
| 因子正交化 | 去除因子間共線性 |
| 機器學習非線性組合 | 捕捉因子交互效應 |

## 10. 最終結論

**台股用公開 price-volume + 法人籌碼數據做量化 alpha 的天花板約 +3~6%/年超額，且不穩定。**

- **動量（momentum）是台股唯一穩定的定價因子**，但僅在低換手率（月頻再平衡）和大型股條件下有效
- **所有 75 個因子在 142 支寬 universe 均未通過 ICIR 0.5 門檻**
- **1/N 等權在台股是極強的基準**，DeMiguel (2009) 結論完全驗證
- **Selection Bias 是量化研究最大陷阱** — 小 universe 的亮眼結果在寬 universe 消失

要真正突破，需要：即時/獨家數據、基本面因子、或根本性地降低交易成本。
