- 60d ICIR 0.646 表明 acceleration 有長期預測力

## 4. 誠實結論

### 策略可行但邊緣

- CAGR +13.8% 接近但未達 15% 門檻
- Sharpe 0.79 勉強通過
- Walk-Forward p = 0.073（不顯著）
- OOS 2025 H1 仍為負

### 之前的結果被高估了

| 項目 | 高估幅度 | 原因 |
|------|---------|------|
| CAGR | ~15pp | 營收公布延遲（look-ahead bias） |
| Sharpe | ~0.8 | 同上 |
| ICIR | ~0.5 | 同上 |
| 年度穩定性 | 3 年 | 2020-2022 實際為負 |

### 正面

1. revenue_acceleration ICIR 0.476（修正後仍強）
2. revenue_yoy t = 3.21（仍通過 Harvey 門檻）
3. 策略正報酬（+13.8%），只是不如之前認為的那麼多
4. 空頭偵測有效（hedged OOS -7.9% vs baseline -16.2%）

### 下一步建議

1. **改用 revenue_acceleration 作為主因子**（受延遲影響最小）
2. **營收公布日觸發**（每月 10 日後立即交易，縮短延遲）— Phase N 的事件時機層
3. **Paper Trading 是唯一的最終驗證** — 回測已到極限
