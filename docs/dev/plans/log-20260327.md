**修正的 2 個公式**：
- **Sharpe Ratio**：幾何年化分子 / 算術年化分母 → 純算術 `mean/std × sqrt(252)`
- **Sortino Ratio**：只用負報酬 std → 全觀測值 `sqrt(mean(min(r,0)²))`

影響：之前的 Sharpe 偏低約 0.1（保守偏差），Sortino 也偏低。修正後需要重跑 StrategyValidator。
