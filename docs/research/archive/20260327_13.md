## 關鍵結論

### 1. 多因子不一定比單因子好

rev_yoy 單因子 ICIR 0.674 已經非常強。最佳兩因子組合 (rev_yoy + rev_accel) 只比單因子高 **+0.018**（2.7% 提升）。三因子和四因子組合反而更差。

**原因**：四個營收因子高度相關（都來自同一個月營收數據），加入更多因子稀釋了最強因子的信號，沒有帶來新的信息。

### 2. 價格因子是毒藥

任何包含 mom_6m / mom_1m / volatility 的組合都大幅惡化。營收因子和價格因子在台股是**負相關的**（高營收成長的股票近期可能已經漲過 → momentum 為正 → 但 forward return 為正是因為營收持續好，不是因為動量）。

等權混合時，價格因子的噪音會蓋過營收因子的信號。

### 3. 最佳實戰配置

| 選項 | ICIR | 複雜度 | 建議 |
|------|:----:|:------:|------|
| **rev_yoy 單因子** | **0.674** | 最低 | **首選（簡單至上）** |
| rev_yoy + rev_accel (70/30) | 0.692 | 低 | 微幅提升，可考慮 |
| 四因子等權 | 0.572 | 中 | 不建議（增加複雜度但降低 ICIR） |

## 跨類型組合的關鍵發現

### 1. 籌碼面因子是毒藥

**投信（trust_10d）和外資（foreign_20d）加入後 ICIR 大幅下降**：
- trust_10d 拖累 -0.164（從 0.674 降到 0.510）
- foreign_20d 拖累 -0.253（從 0.674 降到 0.421）

**原因推測**：籌碼面數據覆蓋 205 支但只有 51 支與營收數據重疊，交集太小導致 IC 不穩定。而且投信/外資的 cross-sectional IC 本身就很弱（ICIR < 0.12）。

**重要：這與 FinLab 的策略不矛盾。** FinLab 的投信策略成功是因為用**條件篩選**（trust > threshold AND revenue > threshold），不是因子的線性組合。籌碼面作為篩選條件有效，但作為 IC 加權因子無效。

### 2. 融資變化是唯一正向因子

margin_chg（融資餘額 20 日變化率，反向）是唯一能微幅提升 rev_yoy 的因子。邏輯：融資減少 = 散戶出清 = 反向信號（與 FinLab 的散戶反向策略一致）。

但改善僅 **+0.007**（1%），且 margin_chg 只覆蓋 51 支（TW50），可能是 selection bias。

### 3. PE 反向但無加成

pe_ratio ICIR = -0.202（高 PE → 低報酬，即成長股溢價反轉）。加入 rev_yoy 後 ICIR 微降，不互補。

---

## 最終結論

**rev_yoy 單因子仍然是最佳選擇。**

沒有任何跨類型因子能顯著改善 rev_yoy（最佳改善僅 +1%）。原因：
1. 非營收因子本身太弱（ICIR < 0.12）
2. 覆蓋率不足（51~205 支 vs 營收 294 支）
3. 籌碼面適合做條件篩選，不適合做因子加權

**建議**：
- **因子層**：rev_yoy 單因子（ICIR 0.674）
- **策略層**：用投信/融資作為**條件篩選**（FilterStrategy），不作為因子加權成分
- 這正是 revenue_momentum 策略的設計：營收因子選股 + 價格/量條件篩選

## Part 7：多因子組合分層分析（大/中/小市值）

676 支依每日市值代理（close × 20d avg vol）分三等分，每 20 天取樣。

### 全部組合 × 全部市值層

| 組合 | 全部 | 大型 | **中型** | 小型 |
|------|:----:|:----:|:--------:|:----:|
| **rev_yoy（單因子）** | **0.724** | 0.437 | 0.542 | 0.625 |
| **rev_yoy + rev_accel** | **0.732** | 0.403 | **0.628** | 0.626 |
| rev_yoy + accel + new_high | 0.640 | 0.323 | 0.630 | 0.628 |
| all 4 revenue | 0.612 | 0.317 | 0.625 | 0.622 |
| rev_yoy + trust_10d | 0.593 | 0.379 | 0.508 | **0.648** |

### 關鍵發現

1. **rev_yoy + rev_accel 的中型股 ICIR 0.628 是全場最高組合**
   - 比 rev_yoy 單因子中型（0.542）高 +0.086
   - 這是唯一一個組合在某個市值層顯著優於單因子的案例

2. **rev_yoy + trust_10d 的小型股 ICIR 0.648 最高**
   - 投信在小型股上有額外信息（機構介入小型股 = 強信號）
   - 但全市場 ICIR 被大型股拖累（0.593 < 0.724）

3. **大型股所有組合都 < 0.44** — 大型股上營收因子天花板就在這裡

4. **中型股是多因子組合的甜蜜點**
   - rev_yoy + rev_accel 中型 0.628 > 任何單因子的任何市值層
   - 三因子/四因子中型也都 > 0.62

5. **加入更多因子在「全部」和「大型」持續惡化，但中型/小型穩定**
   - 全部：0.724 → 0.732 → 0.640 → 0.612
   - 中型：0.542 → 0.628 → 0.630 → 0.625（穩定）
   - 原因：大型股因子稀釋效果 > 中小型股的互補效果

### 策略啟示

| 策略 | 最佳因子配置 | 預期 ICIR |
|------|------------|:--------:|
| **全市場等權** | rev_yoy 單因子 | 0.724 |
| **中型股專注** | rev_yoy + rev_accel | **0.628** |
| **小型股專注** | rev_yoy + trust_10d | **0.648** |
| 大型股 | rev_yoy 單因子 | 0.437 |

**如果要做市值分層策略**：
- 大型股：rev_yoy 單因子（簡單最好）
- 中型股：rev_yoy + rev_accel 雙因子（互補效果最大）
- 小型股：rev_yoy + trust_10d（投信介入小型股是強信號）
